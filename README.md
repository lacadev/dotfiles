Just to be clear, these assume you are using *Ubuntu*. Go ahead, judge me. I too used to be an Arch user.

## Installation
Run:
```bash
git clone git@github.com:lacadev/dotfiles.git && chmod +x dotfiles/setup.sh && dotfiles/setup.sh
```
